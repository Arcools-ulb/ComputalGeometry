Simplex Range Searching

The game consist to find the triangle containing the most point.

Click on the bouton "Next" to start the game

  • You have point set S (Black) which are the points which will be counted inside triangles
  • You have points set R (Red) which are the points forming the possible triangles
  • You choose 3 points between the set R (Red) to form the best triangle and click on button "Next"
  • You have 3 chances to find the best triangle
When you win a game, the difficulty increase. If you want restart or retry with a new sets. Click on "clear". Have fun and good luck.
